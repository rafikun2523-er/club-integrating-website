About Our Platform

Unified Club Portal is a centralized system for students to explore, join and participate in various university clubs efficiently.

Our Policies

Privacy Policy(PP)

1. User data is kept secure and confidential.
2. Personal information (name, email, department) is protected.
3. Data is used only for platform-related purposes.
4. Users can update or delete their profile anytime.

Event Policy(EP)

1. Events must follow university guidelines.
2. Proper approval is required before publishing events.
3. Members will receive prior event notifications.
4. Event conduct must maintain university discipline.

User Conduct(UC)

1. Respectful behavior is mandatory.
2. No misuse of platform resources is allowed.
3. Offensive content is strictly prohibited.
4. Violation may result in suspension.

Notification Policy(NP)

1. Users will receive event alerts.
2. Important announcements will be notified.
3. Club updates will be shared regularly.
4. Users can manage notification preferences.

Account Registration & Verification Policy (ARVP)

1.Only valid student ID holders can register.
2.Email and student information must be authentic.
3.Fake accounts will be permanently removed

Security & System Usage Policy (SSUP)

1.Users must not attempt hacking or unauthorized access.
2.System misuse will lead to strict action.
3.Regular security updates are maintained
